## UI Patterns
1. Button
- Used to trigger actions, such ad submitting a form, opening a menu, or navigating to another page.
- Can have different styles: primary, secondary, icon buttons, etc.

2. Image Sliders
- A carousel of images that users can swipe or click through
- Can be linear (horizontal) or circular.

3. Pagination
- Allows users to navigate through large sets of content (like pages of a blog or products).
- Typically include placeholders, validation, and lables.

4. Progress Bars
- Visual indicators of progress for a task, like uploading a file or completing steps in a form.
- Can be linear (horizontal) or crucial

5. Input Fields
- Areas where users to navigate through large sets of content (like pages of blog or product).
- Typically includes previous/next buttons or page numbers

6. Chip/Tag
- Small, interactive elements representing iteems, categories, or actions.
- Commonly used for tags, filters or selected items or preferences

7. Toogle
- Switches between two stars, like ON.OFF or ENABLE/DISABLE
- Provides an easy way to change settings or preferences

8. Slider Controls
- Allow users to select a value from a range by dragging a handle
- Common for volume, brightness, or price filters

9. Pickers
- UI components for selecting options, such as dates, times or colors
Examples: Date Picker, Time Picker, Color Picker

10. Accordions
- Collapsible sections that show or hide content when clicked.
- Useful for FAQs, menus, or content-heavy pages

11. Dropdowns
- A list of options that appears when a user clickes or hovers over a field
- Saves space and keeps the UI clean offering multiple choices

12. Cards
- Containers for grouping related information, like product details, blog posts, or user profiles
- Often include images, text, and actions

13. Breadcrumbs
- A navigation aid showing the user's location within a website or app hierarchy
- Helps users understand their path and go easily

14.  Hamburgers (Menu Icon)
- A three-line icon representing a hidden menu
- Common in mobile or responsive designs to save screen space
