## Mood board
- A collection of images, materials, tests and other elements that convey the visual style or concept of a project

## Style Guide
- A document that provides guidelines for our UI design and branding project, including shape, typography, colours, and layouts of UI elements

## Colour Palette
- A selected set of colours used consistently throughout a design to create a cohesive look.

## Colour Models
- Various systems for representing colours. common models include RGB(Red, Green, Blue), CMYK (Cyan, Magenta, Yellow, Black), HSB and HSL (Hue, Saturation, Lightness).

## Hex Code
- A six-digit hexadecimal number used in web design to represent colours.

## Solid Colour
- A single, uniform colour without gradient or patterns

## Gradient
- A gradual transition between two or more colours

## Pattern
- A repetitive design or motif uses as a background or texture

## Fill
- The colour, pattern or texture inside a shape or element 

## Outline 
- The Border or edge of a shape, often defined by a line or stroke
